Está página web foi desenvolvido pela aluna Karmine Geremia do curso de Ciência da Computação (IFSUL), com o auxílio do professor Telmo Júnior da disciplina de Criação de Páginas WEB.

O site tem como objetivo apresentar o relatório técnico sobre o HTML5 através do uso da linguagem de marcação HTML, estilização com CSS e scripts com JavaScript.



Clique aqui para fazer download do relatório técnico.

HTML é uma linguagem usada para publicar conteúdos como vídeos e imagens na WEB. HTML5 é a quinta versão da linguagem HTML e seu objetivo é tornar mais prática a manipulação dos elementos na página, além de fazer o CSS (Cascading Style Sheets) e a linguagem de programação dinâmica Javascript terem uma melhor performance.

Algumas tags do HTML5 foram modificadas ou descontinuadas, além de algumas criadas. Foi adicionado as tags: canvas, embed, audio, datalist, nav etc... Neste site foi utilizado em destaque a tag nav para criar uma barra de navegação e um menu responsivo através de CSS e Javascript, além de usar um vídeo como background do site apenas com CSS.

O HTML5 é o suporte de dispositivos móveis muito mais prático atualmente ao ser capaz de reduzir travamentos. Outro ponto que vale ressaltar é que o Adobe Flash não é mais necessário usar.
